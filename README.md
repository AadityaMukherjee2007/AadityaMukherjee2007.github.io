# Aaditya Mukherjee — Portfolio

Personal portfolio website showcasing my projects, skills, and achievements.

**Live:** [aadityamukherjee2007.github.io](https://aadityamukherjee2007.github.io)

## Built With
- HTML
- CSS
- JavaScript (vanilla)

## Features
- Responsive design (mobile, tablet, desktop)
- Project carousel with YouTube embeds
- Scroll-triggered animations
- Contact form (via Formspree)
- SEO & Open Graph meta tags

## Sections
- About Me
- Tech Stack (Languages, Hardware, Tools, Domains)
- Projects (carousel)
- Academic Background & Certifications
- Extracurricular Activities
- What I'm Working On
- Contact

## Setup
No build step required — just open `index.html` in a browser or deploy to GitHub Pages.
